Comodojo/zip documentation
==========================

.. _PHP ZipArchive: http://php.net/manual/en/class.ziparchive.php

This library simplifies and automates the management of zip files using the `PHP ZipArchive`_.

It includes the ``\Comodojo\Zip\ZipManager`` class to manage multiple Zip files at once (extract, add, remove, merge).
